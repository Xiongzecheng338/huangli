## ⚙️ 修炼法门 · 快速上手

### 🚀 一、晨起观历，定一日之纲

每日晨起，先入天机黄历，观当日卦象、详读宜忌，一日行事皆以此为纲，不犯禁忌，顺时而为。

```bash
# 打开天机黄历
https://badhope.github.io/yellow-calendar/

# 或使用本地修炼
git clone https://github.com/badhope/yellow-calendar.git
cd yellow-calendar
# 用浏览器打开 index.html 即可
```

### 🎯 二、大事择吉，借天地之力

凡有动土开市、乔迁出行、婚嫁缔约等大事，必先查本历择定吉时吉方，借天时地利，助万事顺遂。

**操作步骤:**
1. 进入平台 → 点击【择吉日】
2. 选择事件类型 (嫁娶/出行/开业/入宅/动土/祭祀/剃头/求医)
3. 查看当月吉日推荐 (评分 80+ 为吉)
4. 点击吉日查看详情
5. 保存或分享吉日

### 🔮 三、心有困惑，卜卦定方向

若遇前路迷茫、抉择难断之时，可借本平台卦象玄机，卜问吉凶，于混沌中寻得方向，于迷茫时定下心神。

**卜卦心法:**
```
1. 净手焚香，静心凝神
2. 默念所问之事三遍
3. 点击【每日一签】或【六爻起卦】
4. 解读卦象玄机
5. 依卦行事，顺势而为
```

### 📚 四、常修经典，养自身气运

所谓"福地福人居"，外借风水之力，内修心性之德。常读国学经典、循养生之道，内外兼修，方是福泽绵长的根本。
